## 这本书是什么

《C和指针》是公认的 C 语言进阶经典，但原书写于 1997 年，部分内容已过时，且默认读者具备大量隐性知识——很多关键概念一句话带过，初学者读起来容易卡住。

这个项目用 AI 将原书 18 章重组为 **13 个模块、29 课**的模块化教材。每一课都是一个独立的 Markdown 文件，打开就能学，不需要翻书、搜博客、看视频。

## 适合谁读

- 了解 C 基础语法（数据类型、控制流、函数），但对**指针高级用法**缺乏实战经验
- 想系统掌握用 C 实现数据结构（链表、树、堆栈、队列）
- 目标方向是 **Linux 系统编程**，需要扎实的底层基础

## 本书特色

**每个概念四段式讲解**：`是什么 → 为什么重要 → 代码演示 → 易错点`，不跳步。

**完整可运行代码**：每个示例直接复制就能编译运行，不是片段。复制、粘贴、`gcc`、运行——四步搞定。

**正反对比**：每个概念都配 ❌ 错误写法 + ✅ 正确写法，让你先看到坑，再学会避开。

**三档深度**：
- `[BEGINNER]` — 入门补充，帮你补上"书里默认你会"的前置知识
- 正文 — 实用级，读完就能独立运用
- `⭐ 深入` — 底层原理，想深挖就读，跳过也不影响后续

**联网检索 + 过时标注**：AI 写每一课时都会搜索最新资料，原书过时的内容标注 `🔄 原书更新说明`，给出现代替代方案。

**七维勘验**：每课写完后经过系统化审查——对照原书核查事实、实际编译运行每个代码示例、验证图表逻辑、检查术语一致性。不是"AI 生成就完事了"。

## 编译环境

书中所有代码示例使用 C99 标准，编译命令：

```bash
gcc -std=c99 -Wall -Wextra -g -o example example.c
./example
```

开启 `-Wall -Wextra` 是为了让编译器帮你抓住尽可能多的潜在问题——这也是本书推荐的开发习惯。
