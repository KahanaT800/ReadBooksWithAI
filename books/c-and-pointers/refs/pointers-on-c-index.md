## 使用说明

1. 写课时根据课程主题，在「关键概念」列或「概念反查表」中匹配
2. **优先使用 epub 文本**：直接用 Read 工具读取 `epub-extracted/chNN.txt` 对应行号范围
3. epub 文本中的行号即为推荐引用位置（见「epub 行号」列）
4. 仅当 epub 文本不够清晰时（如图表、代码排版丢失），再用 PDF 补充
5. 读取 PDF 时需将书页 +21 转为 PDF 页码，每次不超过 20 页
6. 下表找不到匹配时，用网络搜索补充
