# 《C和指针》索引

> 源文件：`refs/pointers-on-c.pdf`
> PDF 页码偏移：**PDF 页 = 书页 + 21**（例如书页 91 → PDF 页 112）

## 使用说明

1. 写课时根据课程主题，在「关键概念」列或「概念反查表」中匹配
2. **只读取「推荐页」列标注的页码**（为书页），不要读整章
3. 推荐页不够时，再扩展到「页码范围」的完整区间
4. 读取 PDF 时需将书页 +21 转为 PDF 页码
5. 下表找不到匹配时，用网络搜索补充
