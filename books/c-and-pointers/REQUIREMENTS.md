# REQUIREMENTS.md — 《C和指针》

## 原书信息

| 字段 | 值 |
|------|---|
| 书名 | C和指针 (Pointers on C) |
| 作者 | Kenneth A. Reek |
| 原书章数 | 18 章 |
| 学科领域 | C 语言编程 |

## 目标读者

- **背景**：了解 C 语言基础语法（数据类型、控制流、函数等），但对指针高级用法和用 C 实现数据结构缺乏实战经验
- **目标**：扎实掌握指针和数据结构，为 Linux 系统编程打基础
- **薄弱环节**：复杂指针操作（多级指针、函数指针、指针与数组的关系）、用 C 实现数据结构（链表、树等）、动态内存管理

## 深度设定

```yaml
target_depth:
  beginner: minimal      # 基础语法不重复讲，偶尔提醒关键前置概念
  standard: full         # 每个概念完整展开，讲透为止
  advanced: moderate     # 系统编程相关底层原理适度展开，纯标准规范细节控制篇幅
```

## 特别要求

- **强化**：指针及之后的所有核心内容全面展开，包括但不限于：
  - 指针基础与指针运算
  - 多级指针
  - 函数指针与回调
  - 指针与数组的关系
  - 动态内存管理（malloc/calloc/realloc/free）
  - 字符串与指针
  - 结构体与指针
  - 链表、树等数据结构的 C 实现
  - 预处理器高级用法
  - 文件 I/O
- **弱化**：前几章 C 基础回顾（数据类型、基本语法、运算符、控制流）快速带过，不做完整展开
- **其他**：面向 Linux 系统编程方向，示例和练习尽量贴近系统编程场景

## 开发环境

- 语言：C
- 编译器：GCC（WSL2 环境）
- 平台：WSL2 (Linux on Windows)
- IDE：VS Code + Remote WSL
- C 标准：C99（兼顾现代特性和广泛兼容性）
- 编译命令示例：`gcc -std=c99 -Wall -Wextra -g -o output source.c`
